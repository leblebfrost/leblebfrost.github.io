Positioning

The position property specifies the type of positioning method used for an element (static, relative, fixed, absolute or sticky).


Tag Name Description
<static> Static Default value. Elements render in order, as they appear in the document flow
<absolute> Absolute The element is positioned relative to its first positioned (not static) ancestor element
<fixed> Fixed The element is positioned relative to the browser window
<relative> Relative The element is positioned relative to its normal position, so "left:20px" adds 20 pixels to the element's LEFT position
<initial> Initial Sets this property to its default value.
<Inherit> Inherit Inherits this property from its parent element.
